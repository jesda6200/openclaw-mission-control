# 2026-03-27 — Préférence GitHub

- L'utilisateur donne un feu vert permanent pour créer un dépôt GitHub privé et y publier du contenu **si la demande explicite porte sur une publication GitHub / création de repo / push GitHub**.
- Cette préférence vaut comme autorisation préalable pour les futures demandes de dépôt GitHub privé, sans redemander de confirmation séparée, tant que l'action reste cohérente avec la demande en cours.
- En cas d'ambiguïté sur le repo cible, l'owner, ou le contenu à publier, demander seulement le minimum nécessaire.
