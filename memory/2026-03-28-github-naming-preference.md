# 2026-03-28 — Préférence nommage GitHub

- Quand l’utilisateur demande de publier sur son GitHub sans préciser de nom de dépôt, utiliser automatiquement le compte GitHub connecté.
- Inventer un nom de dépôt pertinent si nécessaire, sans redemander le nom.
- Si le projet existe déjà ou qu’un nom est déjà pris / ambigu, suffixer avec `-v2`, `-v3`, etc.
- Cette préférence s’applique aux futures demandes de création de repo / push GitHub, sauf indication contraire explicite.
